# Usage
<!-- usage -->
```sh-session
$ npm install -g csvDrill
$ csvDrill COMMAND
running command...
$ csvDrill (-v|--version|version)
csvDrill/0.0.0 linux-x64 node-v10.15.3
$ csvDrill --help [COMMAND]
USAGE
  $ csvDrill COMMAND
...
```
